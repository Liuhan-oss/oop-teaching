# OOP智慧教学平台

面向对象程序设计智能教学辅助系统

## 项目简介

OOP智慧教学平台是一个融合人工智能技术的C++教学辅助系统，支持教师教学管理与学生自主学习。

## 核心功能

### 教师端
- 📊 数据总览：实时统计教学数据
- 📈 班级学情：多维度分析班级掌握度
- 📝 作业管理：支持多题型发布，AI智能批改
- 🎯 知识点管理：知识图谱展示掌握度分布
- 📚 课件管理：PPT/PDF/Word在线预览
- ❓ 题库管理：AI生成题目，智能分析难度

### 学生端
- 🏠 学习首页：学习统计、快捷入口
- 🌳 知识图谱：树状结构，颜色标识掌握度
- ✍️ 在线作业：AI批改，即时反馈
- 📖 在线题库：按知识点练习
- 🎁 个性化推荐：AI分析薄弱点，推荐学习资源
- 📊 学习分析：雷达图、章节分析
- 💬 智能助手：24小时在线答疑

## 技术栈

- 后端：Flask + SQLite
- 前端：HTML5 + CSS3 + JavaScript
- AI：智谱AI GLM-4-Flash
- 可视化：ECharts
